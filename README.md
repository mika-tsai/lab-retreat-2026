# 2026 International Epigenetics Symposium

Official event website for the 2026 International Epigenetics Symposium and Annual Meeting of the Taiwan Epigenetics Society.

- Date: October 16-17, 2026
- Venue: Shangri-La Far Eastern, Tainan
- Organizer: College of Medicine, National Cheng Kung University and Taiwan Epigenetics Society

## Website

This is a static website intended for GitHub Pages. Registration submissions are sent to a separately deployed Google Apps Script web application.

## Registration backend

The Apps Script source is stored in `google-apps-script/Code.gs`. After modifying it, create a new deployment version in Google Apps Script for the changes to take effect.
